# Análise de Vendas com Python

Projeto de análise de dados voltado a indicadores de negócio, com foco em tratamento, agregação e leitura executiva de métricas.

## Objetivo
Transformar dados brutos de vendas em informações úteis para acompanhamento de faturamento, performance por categoria e evolução mensal.

## Stack
- Python
- pandas
- matplotlib

## Como executar
```bash
pip install -r requirements.txt
python main.py
```

## O que o projeto gera
- CSV com vendas tratadas
- CSV com faturamento mensal
- CSV com resumo por categoria
- Gráfico PNG com a evolução do faturamento
